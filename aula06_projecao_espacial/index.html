O que é projeção espacial?

Aplicar o modelo treinado sobre toda a área de estudo para gerar um mapa contínuo de adequabilidade.

Modelo treinado Rasters bioclimáticos (Random Forest) + (WorldClim - Brasil) | | └─────────┬───────────────┘ ↓ Para CADA pixel do mapa: "Qual a probabilidade de presença dadas as condições climáticas?" ↓ MAPA DE ADEQUABILIDADE

Predição pixel a pixel

Como funciona

Para cada pixel, o modelo recebe as variáveis bioclimáticas e retorna uma probabilidade:

Pixel (lat=-25.4, lon=-49.3): ┌─────────────────────────────────────┐ │ BIO1=16.2 BIO4=4100 BIO12=1400 │ └────────────────┬────────────────────┘ ↓ Random Forest (500 árvores votam) ↓ Probabilidade = 0.82 (82%)

Repetido para cada pixel válido do Brasil (~2 milhões).

Predição pixel a pixel

Dados de entrada

  • Cada variável = 1 arquivo GeoTIFF (ex: bio1_brazil.tif)
  • Mesmo sistema de coordenadas e resolução
  • Variáveis de temperatura (BIO1-11): divididas por 10
  • Pixels sem dados (oceano) = NoData (-9999)
As variáveis da projeção devem ser exatamente as mesmas do treinamento, na mesma escala.

O threshold é o ponto de corte entre "adequado" e "inadequado".

Threshold (Limiar de Corte)

O que é

Probabilidade ≥ threshold → ADEQUADO (presença) Probabilidade < threshold → INADEQUADO (ausência)
A escolha do threshold muda significativamente o mapa e a área calculada. Não existe valor "certo" universal.

Conservação: threshold baixo (capturar toda área potencial).
Plantio comercial: threshold alto (só alta adequabilidade).

Cálculo de Área

Do pixel ao km²

Resolução: 2.5 min de arco ≈ 5 km Área de 1 pixel ≈ 5 km x 5 km ≈ 25 km² Área adequada = n. pixels adequados x área do pixel
Exemplo: 40.000 pixels adequados × 25 km² ≈ 1.000.000 km²

Interpretação dos Mapas

O que o mapa NÃO é

O mapa de adequabilidade não é um mapa de presença real! Mostra onde as condições climáticas são compatíveis.

A espécie pode não estar presente em áreas adequadas por:

  • Barreiras de dispersão (rios, montanhas)
  • Competição com outras espécies
  • Desmatamento ou uso do solo
  • Variáveis não incluídas no modelo
