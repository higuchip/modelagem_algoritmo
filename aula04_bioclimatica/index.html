O que são variáveis bioclimáticas?

Variáveis bioclimáticas são resumos estatísticos do clima que têm significado biológico direto para as espécies.

Em vez de usar dados brutos de temperatura e precipitação mês a mês, usamos variáveis que capturam padrões relevantes para a vida dos organismos.

Analogia: boletim escolar

Imagine que você quer avaliar o desempenho de um aluno:

Dados brutos:
  • Nota de cada prova
  • Nota de cada trabalho
  • 12 meses x várias disciplinas
  • Centenas de números!
Resumos úteis:
  • Média anual
  • Nota mais alta / mais baixa
  • Variação entre bimestres
  • Poucos números, muita informação!

Variáveis bioclimáticas fazem o mesmo: resumem o clima em números biologicamente significativos.

Por que "bio" + "climáticas"?

Não são qualquer resumo do clima. São calculadas pensando no que importa para os seres vivos:

  • Temperatura média anual: define o regime térmico geral
  • Precipitação do mês mais seco: indica estresse hídrico
  • Sazonalidade da temperatura: indica estabilidade climática
Exemplo florestal: a Araucaria angustifolia precisa de invernos frios. A variável "temperatura mínima do mês mais frio" (BIO6) capta exatamente essa exigência.

As 19 variáveis bioclimáticas

O padrão internacional define 19 variáveis, divididas em dois grupos:

Variáveis de temperatura (BIO1-BIO11)

Código Variável
BIO1 Temperatura média anual
BIO2 Amplitude térmica diária média
BIO3 Isotermalidade (BIO2/BIO7 x 100)
BIO4 Sazonalidade da temperatura (desvio padrão x 100)
BIO5 Temperatura máxima do mês mais quente
BIO6 Temperatura mínima do mês mais frio
BIO7 Amplitude térmica anual (BIO5 - BIO6)
BIO8 Temperatura média do trimestre mais chuvoso
BIO9 Temperatura média do trimestre mais seco
BIO10 Temperatura média do trimestre mais quente
BIO11 Temperatura média do trimestre mais frio

Variáveis de precipitação (BIO12-BIO19)

Código Variável
BIO12 Precipitação anual
BIO13 Precipitação do mês mais chuvoso
BIO14 Precipitação do mês mais seco
BIO15 Sazonalidade da precipitação (coeficiente de variação)
BIO16 Precipitação do trimestre mais chuvoso
BIO17 Precipitação do trimestre mais seco
BIO18 Precipitação do trimestre mais quente
BIO19 Precipitação do trimestre mais frio

Relação entre as variáveis

Dados mensais de temperatura e precipitação | v Cálculos estatísticos (médias, extremos, variação) | v 19 variáveis bioclimáticas (BIO1 a BIO19) | v Mapas em formato raster (grid de pixels) Cada pixel = valor da variável naquele local

Cada variável é um mapa onde cada pixel contém o valor climático correspondente àquela localização geográfica.

De onde vêm esses dados?

Existem duas bases de dados principais de variáveis bioclimáticas:

WorldClim

WorldClim (worldclim.org) é a base de dados climáticos mais utilizada em modelagem de distribuição de espécies.
  • Fonte: dados de estações meteorológicas interpolados espacialmente
  • Cobertura: global
  • Resolução: de ~1 km a ~340 km (vários níveis)
  • Período atual: médias de 1970-2000
  • Projeções futuras: disponíveis para diferentes cenários climáticos (SSPs)

Fick & Hijmans (2017). WorldClim 2: new 1-km spatial resolution climate surfaces for global land areas.

CHELSA

CHELSA (chelsa-climate.org) é uma alternativa mais recente, com melhor tratamento de áreas montanhosas.
  • Fonte: dados de reanálise atmosférica + estações
  • Diferencial: considera efeitos orográficos (relevo) na precipitação
  • Resolução: ~1 km
  • Período atual: 1981-2010
No TAIPA: utiliza dados do WorldClim, a base mais amplamente utilizada em estudos de SDM.

Como funciona na prática?

Ponto de ocorrência: Mapa bioclimático (raster): lat: -27.5, lon: -50.3 BIO1 = temperatura média anual | | v v O sistema "consulta" o pixel do mapa naquela coordenada | v Resultado: BIO1 = 16.2 °C naquele ponto

Para cada ponto (presença ou pseudo-ausência), o TAIPA extrai o valor de todas as 19 variáveis bioclimáticas. Isso cria a tabela de dados que alimenta o Random Forest.

A tabela de dados para o modelo

Ponto Classe BIO1 BIO2 BIO12 ...
Presença 1 1 16.2 10.5 1520 ...
Presença 2 1 15.8 11.0 1480 ...
Pseudo-aus. 1 0 22.1 9.3 1100 ...
Pseudo-aus. 2 0 20.5 8.7 950 ...

Esta tabela é a entrada do Random Forest: ele aprende quais combinações de BIO1, BIO2, ..., BIO19 distinguem presença (1) de ausência (0).

Por que selecionar variáveis?

Se temos 19 variáveis disponíveis, por que não usar todas?

Problema 1: Redundância

Muitas variáveis bioclimáticas medem coisas muito parecidas:

Exemplo:
  • BIO1 = Temperatura média anual
  • BIO10 = Temp. do trimestre mais quente
  • BIO11 = Temp. do trimestre mais frio

Essas três são altamente correlacionadas!

Analogia:

É como pedir em um formulário:

  • Sua idade?
  • Seu ano de nascimento?
  • Há quantos anos você nasceu?

São três perguntas para a mesma informação!

Problema 2: Overfitting

Overfitting = o modelo "decora" os dados de treino em vez de aprender padrões reais.

Usar variáveis demais aumenta o risco de overfitting:

  • Com muitas variáveis, o modelo encontra padrões espúrios (falsos)
  • O modelo funciona bem nos dados de treino, mas falha em dados novos
  • A projeção para cenários futuros fica menos confiável

Regra prática: quanto mais simples o modelo (menos variáveis), mais generalizável ele tende a ser.

Problema 3: Interpretabilidade

Um modelo com poucas variáveis é mais fácil de interpretar:

19 variáveis:

"A espécie depende de uma combinação complexa de BIO1, BIO2, BIO3, BIO4, BIO5..."

Difícil de explicar e publicar!

4-6 variáveis:

"A espécie é limitada principalmente pela temperatura mínima no inverno e pela precipitação no período seco."

Claro e ecologicamente interpretável!

O problema da multicolinearidade

O principal motivo técnico para selecionar variáveis:

Multicolinearidade ocorre quando duas ou mais variáveis preditoras são altamente correlacionadas entre si.

O que é correlação?

A correlação mede o quanto duas variáveis "andam juntas":

Correlação (r) Significado Exemplo
+1.0 Perfeitamente positiva Quando uma sobe, a outra sempre sobe
+0.8 Forte positiva BIO1 vs. BIO10 (temperaturas médias)
0.0 Sem correlação Variáveis independentes
-0.8 Forte negativa BIO4 vs. BIO3 (sazonalidade vs. isotermalidade)
-1.0 Perfeitamente negativa Quando uma sobe, a outra sempre desce

Por que a multicolinearidade é um problema?

Se incluímos duas variáveis altamente correlacionadas (ex: |r| > 0.7), o modelo não consegue distinguir qual delas é realmente importante.
  • A importância das variáveis fica distorcida (dividida entre as correlacionadas)
  • O modelo pode atribuir peso a uma variável por acaso, não por relevância ecológica
  • Projeções futuras ficam instáveis: pequenas mudanças nos dados geram resultados muito diferentes

Exemplo concreto

Veja os dados de 5 localidades no Sul do Brasil:

Local BIO1 (Temp. média anual) BIO10 (Temp. trim. quente)
Lages - SC 15.2 °C 19.8 °C
Curitiba - PR 16.5 °C 20.9 °C
Florianópolis - SC 20.1 °C 24.5 °C
Londrina - PR 21.0 °C 25.2 °C
Foz do Iguaçu - PR 21.8 °C 26.0 °C
Perceba o padrão: quando BIO1 sobe, BIO10 sempre sobe junto. A diferença entre elas é quase constante (~4-5 °C). Sabendo uma, já sabemos a outra. Usar as duas no modelo é redundante!

Correlação entre elas: r = 0.95. Devemos manter apenas uma.

Regra prática

Limiar comum: remover uma de cada par de variáveis com correlação |r| > 0.7

Qual das duas remover? Critérios:

  • Relevância ecológica: manter a que faz mais sentido biológico para a espécie
  • Importância no modelo: manter a que o modelo considera mais importante
  • Generalidade: preferir variáveis mais gerais (BIO1) às derivadas (BIO10)

Métodos de seleção de variáveis

Como decidir quais variáveis manter?

1. Matriz de correlação

Calcular a correlação entre todos os pares de variáveis:

BIO1 BIO2 BIO4 BIO12 BIO14 BIO1 [ 1.00 -0.32 0.45 -0.15 0.28] BIO2 [-0.32 1.00 0.78 0.10 -0.05] BIO4 [ 0.45 0.78 1.00 -0.20 0.12] BIO12 [-0.15 0.10 -0.20 1.00 0.85] BIO14 [ 0.28 -0.05 0.12 0.85 1.00]

Identificar pares com |r| > 0.7: BIO2-BIO4 (0.78) e BIO12-BIO14 (0.85).

Para cada par altamente correlacionado, remover uma das variáveis.

2. VIF (Variance Inflation Factor)

O VIF mede o quanto cada variável é explicada pelas outras. Quanto maior o VIF, maior a redundância.
VIF Interpretação Ação
1 Sem colinearidade Manter
1 - 5 Colinearidade moderada Aceitável
> 5 Colinearidade alta Remover
> 10 Colinearidade muito alta Remover com certeza

Processo iterativo: calcular VIF, remover a variável com maior VIF, recalcular, repetir até todas terem VIF < 5. No TAIPA, o threshold padrão é VIF = 5.

3. Importância do modelo (Random Forest)

O Random Forest calcula a importância de cada variável automaticamente:

  • Treinar o modelo com todas as variáveis
  • Verificar quais variáveis mais contribuem para as decisões das árvores
  • Remover as variáveis com baixa importância

Este método é complementar: primeiro remover colinearidade (correlação/VIF), depois usar a importância para refinar.

Fluxo recomendado de seleção

19 variáveis bioclimáticas (BIO1 a BIO19) | [1] Remover por correlação (|r| > 0.7) | ~8-10 variáveis restantes | [2] Verificar VIF (remover VIF > 5) | ~5-8 variáveis restantes | [3] Treinar modelo e verificar importância | 4-6 variáveis finais (conjunto otimizado)

Variáveis-chave para espécies florestais

Na prática, algumas variáveis são frequentemente mais relevantes para espécies florestais no Brasil:

Variáveis frequentemente importantes

Temperatura:
  • BIO1 - Temp. média anual
  • BIO4 - Sazonalidade da temp.
  • BIO6 - Temp. mínima do mês mais frio
Precipitação:
  • BIO12 - Precipitação anual
  • BIO14 - Precip. do mês mais seco
  • BIO15 - Sazonalidade da precip.

Isso NÃO significa que você deve usar sempre essas! Cada espécie tem suas particularidades.

Exemplos práticos

Espécie Variáveis-chave Por quê?
Araucaria angustifolia BIO6, BIO4, BIO12 Precisa de frio no inverno e chuva bem distribuída
Euterpe edulis BIO14, BIO1, BIO15 Sensível à seca, precisa de umidade constante
Handroanthus albus BIO1, BIO12, BIO4 Distribuição ampla, limitada por temperatura e chuva geral
Dica: ao interpretar os resultados, pergunte-se: "faz sentido ecológico que essa variável seja importante para minha espécie?"

Variáveis bioclimáticas e mudanças climáticas

A seleção de variáveis também é importante para projeções futuras:

SSPs: cenários de emissões

SSPs (Shared Socioeconomic Pathways) são cenários que projetam como o clima será no futuro, dependendo das ações humanas.
Cenário Descrição Aquecimento até 2100
SSP1-2.6 Sustentável, baixas emissões ~1.8 °C
SSP2-4.5 Intermediário ~2.7 °C
SSP3-7.0 Alta rivalidade regional ~3.6 °C
SSP5-8.5 Alta emissão, sem mitigação ~4.4 °C

Para cada cenário, as 19 variáveis bioclimáticas são recalculadas com os valores projetados de temperatura e precipitação.

Por que isso importa para a seleção?

Atenção: variáveis que são estáveis hoje podem mudar drasticamente no futuro, e vice-versa.
  • O modelo treinado com dados atuais será aplicado a mapas futuros
  • Usar variáveis com sentido ecológico claro torna a projeção mais confiável
  • Variáveis selecionadas por pura correlação estatística podem gerar projeções irreais
No TAIPA: após treinar o modelo com dados atuais, você poderá projetá-lo para cenários futuros usando as mesmas variáveis selecionadas.

Prática no TAIPA

Agora vamos aplicar o que aprendemos!

Etapa 3 do TAIPA: Análise de Variáveis Bioclimáticas

Passo a passo no TAIPA

  1. 1 Verificar que as etapas anteriores estão completas (ocorrências + pseudo-ausências)
  2. 2 Avançar para a etapa "Análise Bioclimática" (as 19 variáveis já vêm selecionadas)
  3. 3 Definir os parâmetros: threshold de VIF (padrão: 5) e de correlação (padrão: 0.7)
  4. 4 Clicar "Extrair Valores Bioclimáticos" - o TAIPA consulta os mapas WorldClim para cada ponto
  5. 5 Analisar a matriz de correlação e o gráfico de VIF gerados automaticamente
  6. 6 Usar a seleção automática (Método Robusto recomendado) para obter o conjunto final de variáveis

O que o TAIPA faz automaticamente

Extração:
  • Consulta os rasters WorldClim
  • Extrai os valores de BIO1-BIO19 para cada ponto de presença e pseudo-ausência
  • Monta a tabela de dados completa
Seleção (Método Robusto):
  • Calcula correlação entre todas as variáveis
  • Remove redundâncias (|r| > 0.7)
  • Calcula VIF e elimina colineares (VIF > 5)
  • Retorna o conjunto final otimizado

O que observar nos resultados

  • Na matriz de correlação: quais variáveis formam blocos de alta correlação?
  • No gráfico de VIF: quais variáveis ultrapassam o limiar de 5?
  • As variáveis selecionadas cobrem tanto temperatura quanto precipitação?
  • As variáveis fazem sentido ecológico para sua espécie?
Dica: use a opção "Visualizar Layer" para ver o mapa de cada variável com seus pontos sobrepostos. Isso ajuda a entender visualmente o que cada variável representa na sua área de estudo.

O que aprendemos hoje

  • Variáveis bioclimáticas são resumos do clima com significado biológico (BIO1 a BIO19)
  • Os dados vêm de bases como WorldClim e CHELSA (o TAIPA usa WorldClim)
  • O TAIPA extrai os valores das variáveis para cada ponto de presença e pseudo-ausência
  • Não devemos usar todas as 19 variáveis: redundância, overfitting e interpretabilidade
  • Multicolinearidade distorce a importância das variáveis no modelo
  • Usamos correlação e VIF para selecionar 4-6 variáveis informativas e independentes
  • As variáveis selecionadas devem fazer sentido ecológico para a espécie
